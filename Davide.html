Linguaggio Html

Html NON è un linguaggio di programmazione, ma bensi di contenuto.
Viene utilizzato per per strutturare la pagina, servendosi del linguaggio di CSS
e di Javascript per curare lo stile e l'interattività
Nel linguaggio HTML la pagina web viene divisa in due macrosezioni, HEAD e BODY
L'head, contiene riferimenti della pagina che comunicano con il browser, indicando vari aspetti
non visualizzabili dall'interfaccia utente.
Nel body invece, vengono riportate tutto quel genere di informazioni e contenuti che
l'utente riesce a visualizzare quando accede al sito.
Grazie al linguaggo html possiamo avere la possibilità di utilizzare un
un certo numero di funzionalità, tra cui: I titoli, i paragrafi, gli item list, i bottoni, i link etc...
Il body, ovvero la pagina visualizzata dall'utente, si divide a sua volta in tre sezioni: header, main e footer.

Css, a differenza di html non viene utilizzato per inserire contenuto alla pagina,
ma per curare lo stile della stessa, con il fine di renderla più appetibile
all'eventuale visitatore (utente).
Il collegamento del file html con il file css avviene all'interno dell'head della pagina, in modo che sia eseguito
prima dell'esecuzione del contenuto, dando al nostro sito, l'impatto visivo che desideriamo
a partire dal primo momento.
Grazie alle funzioni di CSS infatti, riusciamo a curare aspetti estetici e non, come:
Background, colore del testo, hover, allineamento testo etc.

Javascript invece, è un linguaggio di programmazione utilizzato principalmente
per manipolare contenuti grazie a fuonzioni e variabili.
